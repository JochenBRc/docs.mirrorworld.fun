# Javascript SDK

The Mirror World SDK for JavaScript implements a TypeScript-compatible client
that can be used for both in the browser and Node.js.

## Installation

1. Create a developer account on the <span onClick = {()=> {
   window.mixgather.event("developer_dashboard")
   window.open("https://app.mirrorworld.fun")}}>[Developer dashboard]()</span>.
2. Create a project and copy the API Key for your project.
3. Install SDK in your project


```bash
yarn add @mirrorworld/web3.js
```

## Usage

Import the SDK instance to initialize it.

```ts
import { MirrorWorld, ClusterEnvironment } from "@mirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_SECRET_API_KEY",
  env: ClusterEnvironment.testnet, // Can be ClusterEnvionment.mainnet for mainnet
})

async function main() {
  await mirrorworld.login()
  await mirrorworld.getTokens()
  await mirrorworld.getNFTs()
}

main()
```

If you're building a React Application, you can follow
[this guide](/guides/getting-started-with-mirrorworld-sdk) to get started with
the JavaScript SDK.
