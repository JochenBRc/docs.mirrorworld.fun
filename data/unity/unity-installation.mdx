# Unity SDK

The Mirror World SDK for Unity implements a CSharp-compatible client that can be used in unity projects

## Installation
1. Create a developer account the [developer dashboard](https://app.mirrorworld.fun).
2. Create project and copy the API Key and `clientId` for your project.
3. Install SDK in your project

<GithubButton label={"View Github Repository"} repository={"https://github.com/mirrorworld-universe/mirrorworld-sdk-unity"} />

## Import SDK
To the add the SDK in the unity project go the unity package manager window and add the project with the git url

```url
https://github.com/mirrorworld-universe/mirrorworld-sdk-unity.git
```

For more information about unity package window check the link here [Package Manager Window](https://docs.unity3d.com/Manual/upm-ui.html)

## Usage
Import the SDK instance to initialize it.

```cs
using mirrorworld_sdk_unity.Runtime;
using mirrorworld_sdk_unity.Runtime.Models.Request.Authentication;
using mirrorworld_sdk_unity.Runtime.Services.Implementations;
using mirrorworld_sdk_unity.Runtime.Services.Interfaces;

public class ApiManager : MonoBehaviour
{
void Start()
    {
        IMirrorWorldClient client = new MirrorWorldClient(mirrorworld_sdk_unity.Runtime.Environment.Staging, EnvironmentVersion.V1,
            "xdK36g2iXgX05oXsCyvlkq6XvDQuhG1mUG2");
    }
}
```
