# Android API Reference
## Authentication Methods
### StartLogin
Calling this api would popup a dialog, user can finish login flow on it. In which dialog, user can login with third-party authentication like Google, Twitter. Or they can login with their email.
```java
MirrorSDKJava.getInstance().StartLogin(new MirrorCallback() {
    @Override
    public void callback(String result) {
        Log.i("MyApp","Login flow successed!");
    }
});
```

### IsLoggedIn
Checks whether the current user is logged in. You can use this function to judge whether a user needs to start login flow.
```java
MirrorSDKJava.getInstance().IsLoggedIn(new MirrorCallback() {
    @Override
    public void callback(String result) {
        Log.i("MyApp","Is logged in:"+result);
    }
});
```

### SetDebug
Set if use debug mode. In this mode, developer can see details in whole flow.
```java
MirrorSDKJava.getInstance().SetDebug(true);
```

### SetAppID
Set the api key which the current app uses. You can find it when you create a project on our website.
```java
String apiKey = "Your api key";
MirrorSDKJava.getInstance().SetAppID(apiKey);
```

## Wallet Methods
### GetWallet
Get the user's Sol wallet address.
```java
MirrorSDKJava.getInstance().GetWallet(new MirrorCallback() {
    @Override
    public void callback(String result) {
        Log.i("MyApp","my wallet response is xxxx");
    }
});
```

### GetAccessToken
Get access token so that users can visit APIs.  
```java
Activity activity;
MirrorSDKJava.getInstance().GetAccessToken(activity);
```

### QueryUser
Check user's info, then we can get user's base information such as wallet address and so on.  
```java
String email = "youremail@xxx.com";
MirrorSDKJava.getInstance().QueryUser(email, new MirrorCallback() {
    @Override
    public void callback(String s) {
        Log.i("MyApp","User info is:"+s);
    }
});
```

## Marketplace Methods
### FetchSingleNFTDetails

```java

    @Override
    public void callback(String s) {
```

### MintNFT
Mint a new NFT.
```java

String mintSymbol = "Symbol";

    @Override
    public void callback(String s) {

    }
});
```
