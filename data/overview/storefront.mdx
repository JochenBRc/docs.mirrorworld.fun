# Create storefront

1. Prepare your storefront Link：GitHub link

- Download the project from GitHub
  https://github.com/mirrorworld-universe/marketplace-storefront
- Run the code locally
  - yarn install
  - yarn dev
  - preview：
    <img
      src="https://market-assets.mirrorworld.fun/dashboard/storefront.png"
      width="100%"
      height="auto"
    />
- Download your ObjectConfig file on the dashboard and update on your project
  - Visit projects - Marketplace - your_own_project, click 'Download'
    <img
      src="https://market-assets.mirrorworld.fun/dashboard/dashboard2.png"
      width="100%"
      height="auto"
    />
  - Go back to your GitHub project and replace the 'userConfig.json' with your
    download file. {" "}
    <img
      src="https://market-assets.mirrorworld.fun/dashboard/dashboard2.png"
      width="100%"
      height="auto"
    />
    Please make sure to update your configuration userConfig.json

2. There are multiple ways to deploy your storefront：

- Using vercel to deploy Check deployment tutorial - https://vercel.com/docs
- You can choose other deployment methods.
