# Features
These are the features you immediately have accessible to you with the Mirror World Mobile SDK

1. Social Authentication
2. Login with Email/Password

## Marketplace

2. Mint NFTs
3. List NFTs
4. Buy NFTs
5. Cancel Listing
6. Update Listing
7. Transfer NFT
8. Query NFTs by owners

10. Query NFTs by update authorities
