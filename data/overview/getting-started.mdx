# Getting started

Getting started with the Mirror World SDK in your application requires only 3 simple steps:

1. Create a developer account on the [Mirror World Developer Dashboard](https://app.mirrorworld.fun)
2. Create a new project. Copy the API Key and `clientId`.
3. Install the SDK in your project and start using it.
